# 🚴‍♂️ Cyclistic Bike-Share Case Study

A complete data analysis project completed in R as part of the Google Data Analytics Certificate.

## 📌 Project Overview
This project analyzes 12 months of Cyclistic/Divvy bike-share data to understand how casual riders and annual members use the service differently.  
The goal is to provide insights that support a marketing strategy to convert casual riders into annual members.

## 📊 Files in this Repository
- **cyclistic_analysis.Rmd** — Full R Markdown source code  
- **cyclistic_analysis.html** — Knitted final report (open this to view the full analysis)  
- **README.md** — Project summary

## 🛠️ Tools Used
- R  
- tidyverse  
- dplyr  
- lubridate  
- ggplot2  
- R Markdown  

## 🔍 Key Insights
- Members ride more frequently but for shorter durations  
- Casual riders use bikes more on weekends and for leisure  
- Members use bikes more during weekdays and commuting hours  
- Docked bikes are used more by casual riders  
- Marketing efforts should target casual riders with weekend promotions and membership benefits
